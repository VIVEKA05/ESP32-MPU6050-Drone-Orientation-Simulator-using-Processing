# ESP32-MPU6050-Drone-Orientation-Simulator-using-Processing
# ESP32 MPU6050 Drone Orientation Simulator using Processing  This project demonstrates real-time drone orientation visualization using an ESP32 and MPU6050 IMU sensor. The ESP32 calculates Pitch, Roll, and Yaw values and sends them via serial communication to Processing IDE, where a 3D drone model simulates the motion.
